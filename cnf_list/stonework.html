StoneWork

StoneWork is a high-performance, all-in-one routing platform, running cloud both natively & on bare-metal.

Thanks to its modular architecture, StoneWork dynamically integrates all PANTHEON.tech CNF's. Depending on the configuration, StoneWork only required containers will start - preserving a single, high-performance data-plane.

This way, we achieve the best possible resource utilization, unified declarative configuration, and re-use of data paths for packet punting between CNFs. It keeps the integration non-monolithic but dynamic, based on container orchestration, excellent performance, thanks to utilizing the FD.io VPP data plane.
